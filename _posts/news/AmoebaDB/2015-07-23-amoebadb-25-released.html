We are pleased to announce the release of AmoebaDB Release 25

New data in this release

The data set title below links to a full description and list of associated searches and GBrowse tracks.

Genome Sequence

Genome sequence for the following Acanthamoeba strains is available for the first time in AmoebaDB. The data is provided pre-publication by Andrew Jackson, University of Liverpool.
  • Acanthamoeba astronyxis Unknown — Genbank version 2 Feb 2015.
  • Acanthamoeba castellanii Ma — Genbank version 2 Feb 2015.
  • Acanthamoeba culbertsoni A1 — Genbank version 2 Feb 2015.
  • Acanthamoeba lenticulata PD2S — Genbank version 2 Feb 2015.
  • Acanthamoeba lugdunensis L3a — Genbank version 2 Feb 2015.
  • Acanthamoeba mauritaniensis Reich — Genbank version 2 Feb 2015.
  • Acanthamoeba palestinensis 1652 — Genbank version 2 Feb 2015.
  • Acanthamoeba quina vil3 — Genbank version 2 Feb 2015.
  • Acanthamoeba rhydoses Singh — Genbank version 2 Feb 2015.
  • Acanthamoeba sp Galka — Genbank version 2 Feb 2015. In Genbank as A. pearcei.
  • Acanthamoeba sp Incertae sedis — Genbank version 2 Feb 2015. In Genbank as A. royreba.
  • Acanthamoeba sp T4b-type — Genbank version 2 Feb 2015. In Genbank as A. polyphaga
  • Acanthamoeba triangularis SH621 — Genbank version 2 Feb 2015. In Genbank as A. palestinensis.

RNA Sequencing

  • Trophozoite Transcriptome (Nfow) — Transcriptome of Naegleria fowleri ATCC 30863 trophozoites. Zysset-Burri et al..
