--- layout: post title: FungiDB 3.2 released category: [EuPathDB,VEuPathDB,FungiDB] --- FungiDB3.2News
We are pleased to release FungiDB 3.2.  This release include the following new data and features.

New Data:

  • Genomes and annotation (full description and attribution is available in the data set section):

    • Pneumocystis carinii
    • Zymoseptoria tritici
    • Pythium ultimum var. sporangiiferum
    • Pythium irregulare
    • Pythium iwayamai
    • Pythium arrhenomanes
    • Pythium aphanidermatum
    • Pythium vexans
    • Saprolegnia diclina
    • Aphanomyces astaci
    • Aphanomyces invadans
    • Albugo candida
    • Albugo laibachii


  • High throughput sequencing data from drug-resistant and susceptible clinical and environmental isolates of Aspergillus fumigatus.  Sequencing data was aligned to the reference genome A. fumigatus Af293 and single nucleotide polyorphisms were determined. To run searches against this data, visit the data set section for a full description and links to searches.   

  • RNA sequencing data from 3 Phytophthora sojae samples: media grown, cysts, and from infected soybean hypocotyls.  Additional information and links to searches can be viewed in the data set section.  


New feature:
  • Results analysis tools:  results of any searches can now be analyzed for statistical enrichment or Gene Ontology (GO) terms, metabolic pathways or gene product text terms. A detailed description of this new functionality and a video tutorial are also available.


Note: Our bimonthly database releases incorporate new data and correct old data when necessary.  Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits. When search parameters change with a new release, we invalidate (Ã˜) the search and ask you to rerun it. When IDs are updated or removed, we map the old IDs to the new ones, remove the old IDs from your Basket, and leave your Favorites page alone.
