We are pleased to announce the release of FungiDB 35

New data in this release

Genome Sequence and Annotation: these annotated genomes appear for the first time in FungiDB.

  • Cryptococcus gattii VGIV IND107 (Cryptococcus tetragattii IND107) – Farrer RA et al.
  • Cryptococcus gattii VGIII CA1873 (Cryptococcus bacillisporus CA1873) – Farrer RA et al.

Microarray data

  • Transcriptome of Fusarium graminearum during the infection of wheat – Lysoe et al.

Proteomics data

  • Analysis of the Protein Kinase A-Regulated Proteome of Cryptococcus neoformans – Geddes et al.

New features in this release

Dedicated search for Genes with protein products that have post-translational modifications

FungiDB offers a new search Identify Genes based on Post-Translational Modifications, which returns genes whose protein products were experimentally determined to be post-translationally modified. This search is categorized under Search for Genes, Proteomics, Post-Translational Modification.

GO Slim Implementation

GO Slim terms are now indicated in EuPathDB sites in the following ways:
  • The search, Identify Genes based on GO Terms, now includes an GO Subset option for limiting the search to only GO Slim generic terms.
  • GO Terms tables on gene pages now contain a 'GO Subset' column which identifies GO Slim generic terms.
  • The GO enrichment analysis has an option for limiting the analysis to GO Slim terms. This will limit both the background and the gene list of interest in the Fisher's exact test that determines enrichment.

Revigo and Word Cloud access from GO Enrichment results

FungiDB offers a link between our GO Enrichment results and a Word Cloud visualization as well as Revigo, a tool that provides GO term summarization and visualization. The links to Revigo and the Word Cloud now appear as buttons that are right justified above the GO enrichment result table.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary.
