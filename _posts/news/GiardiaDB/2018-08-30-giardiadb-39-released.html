We are pleased to announce the release of GiardiaDB 39

New data in GiardiaDB 39

Proteomics

  • Giardia Assemblage B secretome IEC infection – Secretome of Giardia intestinalis isolates GS, clone H7 (ATCC50581; assemblage B) during interaction with intestinal epithelial cells (IECs). Ma'ayeh et al. This data set has a host component available in HostDB as: Human intestinal cell secretome during Giardia infection
  • Giardia Assemblage A secretome IEC infection – Secretome of Giardia intestinalis isolates WB clone C6 (ATCC30957; assemblage A) during interaction with intestinal epithelial cells (IECs). Ma'ayeh et al. This data set has a host component available in HostDB as: Human intestinal cell secretome during Giardia infection

RNA Sequence

  • Giardia Assemblage A intestine foci transcriptome – Transcriptome from Giardia foci in murine intestine. Pham et al.

New features in GiardiaDB 39

FPKM Floor parameter in RNA-Seq fold change searches

  • RNA Seq fold change searches now offer an FPKM Floor parameter for adjusting the FPKM value of low expressing genes. To avoid misleading fold change values generated by dividing with very small numbers, we set very low FPKM values to a default FPKM floor which represents 10 mapped reads. The new parameter offers a range of floor values for raising or lowering the expression threshold of the search depending on the user's objectives. For more information use the parameter help (blue question mark icon) on the search page or read the detailed RNA seq search help.
