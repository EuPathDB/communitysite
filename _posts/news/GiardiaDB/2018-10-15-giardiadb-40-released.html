We are pleased to announce the release of GiardiaDB 40.
GiardiaDB 40 represents a software and features release.

New features in GiardiaDB 40

New Sense/Antisense RNA seq search

  • RNA Seq searches —
    A new Sense/Antisense search for exploring strand specific transcription is available for any strand specific RNA sequence data set. For each gene, the search compares the fold change in sense reads to the fold change in antisense reads and returns genes that have simultaneous changes in sense vs antisense expression between two samples. Versatile search parameters allow for choosing the direction and magnitude of the change in sense and antisense transcripts. The search result page lists genes and indicates the paired Reference->Comparison samples that meet the criteria, and offers two graphs to help interpret the search results (FPKM vs sample bar graph, antisense FC vs sense FC scatter plot) Sense/Antisense search tutorial.

New Gene page tools

  • Protein alignment tool —
    The Orthologs and Paralogs within EuPathDB table now includes an option to run protein ClustalW alignments on orthologs. Use the 'Protein clustalW' column to include orthologs in the aligment and click 'Run clustalW for selected genes' to initiate the alignment. For more information see this ClustalW tutorial.
  • Protein-protein BLAST tool —
    The Protein features and properties data section now includes an option to BLASTP the protein sequence against 8 data bases including Non-redundant protein sequences (nr), Reference proteins (refseq_protein), UniProtKB/Swiss-Prot (swissprot), Model Organisms (landmark), Patented protein sequences(pat), Protein Data Bank proteins(pdb), Metagenomic proteins(env_nr), and Transcriptome Shotgun Assembly proteins (tsa_nr). For more information see this BlastP tutorial.
  • MitoProt tool —
    The Protein features and properties data section now includes an option to retrieve mitochondrial prediction data from MITOPROT: Prediction of mitochondrial targeting sequences. Clicking the gene page MitoProt tool Submit button initiates a query of the protein sequence in MitoProt and opens a new tab populated with the prediction results. For more information see this MitoProt tutorial.
  • WoLF PSORT tool —
    The Protein features and properties data section now includes an option to retrieve predictions of protein subcellular localization sites from WoLF PSORT based on amino acid sequences. Choose an Organism type and click Submit to initiate a query of the protein sequence in WoLF PSORT and open a new tab populated with the prediction results. For more information see this WoLF PSORT tutorial.

New Galaxy tool and workflow

  • Map Your Proteins to OrthoMCL Groups workflow —
    EuPathDB Galaxy now includes a preconfigured workflow for mapping proteins to OrthoMCL groups. The workflow uses BLAST and the OrthoMCL algorithm to assign your set of proteins to OrthoMCL groups. Access the workflow from the EuPathDB Galaxy home page under 'Get started with pre-configured workflows'. The workflow employs three tools which can be accessed separately in the EuPathDB Galaxy tools menu under EuPathDB OrthoMCL Tools. This workflow duplicates and replaces the 'Assign your proteins to groups' function in the Tools menu of the OrthoMCL site. For more information see this OrthoMCL proteome upload tutorial.
  • RNA-Seq to EuPathDB export tool —
    EuPathDB Galaxy now includes a tool for exporting the results of your RNA seq analysis to any EuPathDB site. The tool transfers bigwig and FPKM text files to your private My Data Sets page on EuPathDB. From the My Data Sets page the files are available for GBrowse visualization and the Search Strategy system. For more information see this RNA Seq to EuPathDB export tool tutorial.


No new data in GiardiaDB 40

Note: Our bimonthly database releases incorporate new data and correct old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits.
