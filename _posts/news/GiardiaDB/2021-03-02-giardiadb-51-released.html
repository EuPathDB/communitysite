New data in {{project}} 51

EBI Pipeline

{{project}} 51 represents the first release that employs the EBI pipeline for analyzing genomes and RNA-Seq data. ({{project}} Analysis Methods) All genomes were run through the EBI pipeline, updating all analyses such as InterproScan and repeat identification. While much of the EBI pipeline is the same as our previous in-house pipeline, these differences are noted:
  • Expression values calculated from RNA-Seq data are now presented as TPM (Transcripts Per Kilobase Million) instead of FPKM (Fragments Per Kilobase Million) or RPKM (Reads Per Kilobase Million). Learn more
  • Signal peptide predictions are now made with SignalP-4.1, updated from SignalP-3.0.

New features in {{project}} 51

  • BLASTbeta tool – a new BLAST tool with multi-BLAST capabilities. Input a single sequence or several fasta-formatted sequences at once. Our tool uses NCBI BLAST to find genes, genomic sequences, ESTs, or PopSet Sequences that have BLAST similarity to your input sequence.
  • Usage and performance metrics for VEuPathDB sites are updated monthly in a new online document.
