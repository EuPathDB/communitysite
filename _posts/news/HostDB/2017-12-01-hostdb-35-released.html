We are pleased to announce the release of HostDB 35

New data in this release

RNA Sequencing

  • Transcriptomes of 4 M. musculus cell types during infection with T. gondii — M. musculus cortical neurons, astrocytes, skeletal muscle cells and fibroblast transcriptomes during infection with T. gondii strain NTE. Swierzy et al.

New features in this release

GO Slim Implementation

GO Slim terms are now indicated in EuPathDB sites in the following ways:
  • The search, Identify Genes based on GO Terms, now includes an GO Subset option for limiting the search to only GO Slim generic terms.
  • GO Terms tables on gene pages now contain a 'GO Subset' column which identifies GO Slim generic terms.
  • The GO enrichment analysis has an option for limiting the analysis to GO Slim terms. This will limit both the background and the gene list of interest in the Fisher's exact test that determines enrichment.

Revigo and Word Cloud access from GO Enrichment results

HostDB offers a link between our GO Enrichment results and a Word Cloud visualization as well as Revigo, a tool that provides GO term summarization and visualization. The links to Revigo and the Word Cloud now appear as buttons that are right justified above the GO enrichment result table.

Note: Our bimonthly database releases incorporate new data and correct old data when necessary.
