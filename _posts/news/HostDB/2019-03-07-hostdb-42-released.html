New data in HostDB 42

Genome Sequence and Annotation

  • Bos taurus breed Hereford — The genome sequence and annotation for the cow genome appears for the first time in HostDB. Zimin et al.

Transcriptomics

  • Host cell transcriptome in bovine cells infected with Cryptosporidium parvum —
    Transcriptome of MDBK cells during C. parvum intracellular infection. Reads were mapped to the Bos taurus, breed Hereford reference genome. Widmer lab unpublished.

New features in HostDB 42

  • Updated Alignment Tool on Gene Pages –
    The Clustal Omega alignment tool in the 'Orthologs and Paralogs within EuPathDB' table now accommodates protein, CDS and genomic sequence alignments. To run Clustal Omega, select genes from the table. Then choose the sequence type and initiate the alignment with the 'Run Clutal Omega for selected genes' button. Output formats include Mismatches highlighted, FASTA, PHYLIP, STOCKHOLM, and VIENNA. This tool replaces the 'Retrieve multiple sequence alignment or multi-FASTA' tool.
  • Beta access to JBrowse –
    A beta implementation of the JBrowse Genome Browser, is available from gene pages. Use the JBrowse Genome Browser link under the graphic Shortcuts on the gene page. Example Link Location
  • New Gene Page STRING tool –
    The 'Protein features and properties' section now includes an option to submit the protein sequence to the STRING functional protein association network. Szklarczyk et al.
  • Result Table Improvements –
    Result column configurations and sorting are now saved for each strategy step and will be retained when you come back or when you share a strategy with someone else.
  • New Options for FASTA downloads –
    The header line of FASTA formatted downloads can be configured to contain only the gene ID. The sequence can be configured to appear on a signle line or the standard 60 characters per line.


Note: Our bimonthly database releases incorporate new data and correct old data when necessary.
