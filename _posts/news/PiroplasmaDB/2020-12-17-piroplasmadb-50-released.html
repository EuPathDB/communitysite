New data in {{project}} 50

EBI Pipeline

{{project}} 50 represents the first release that employs the EBI pipeline for analyzing genomes and RNA-Seq data. ({{project}} Analysis Methods) All genomes were run through the EBI pipeline, updating all analyses such as InterproScan and repeat identification. While much of the EBI pipeline is the same as our previous in-house pipeline, these differences are noted:
  • Expression values calculated from RNA-Seq data are now presented as TPM (Transcripts Per Kilobase Million) instead of FPKM (Fragments Per Kilobase Million) or RPKM (Reads Per Kilobase Million). Learn more
  • Signal peptide predictions are now made with SignalP-4.1, updated from SignalP-3.0.

New features in {{project}} 50

  • When revisiting a search page, your previous parameter values are automatically loaded. This is especially useful for the BLAST search which now retains the input sequence.
  • The cytoscape drawing on metabolic pathway record pages now contains:
    • a new search function for finding nodes within the drawing. After running a search, the drawing centers on the nodes which match the search term and highlights them with a bold green border.
    • a "Reset Layout" button that will clear all node paintings as well as reset the node positions and zoom levels.
    • a checkbox for toggling your mouse function between zooming the drawing or scrolling the page.
  • Data set and Organism record pages now contain direct links to our Downloads section where the option is available to download genome-scale sequence and annotation files.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits.
