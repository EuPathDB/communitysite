We are pleased to announce the release of ToxoDB 30

New data in ToxoDB Release 30

Genome Sequence and Annotation:

The following genomes were updated to the current Genbank versions which include new sequence as well as annotation that appears for the first time in ToxoDB.
  • Toxoplasma gondii strain ARI. – Genbank version March 14, 2016
  • Toxoplasma gondii strain CatPRC2 – Genbank version March 23, 2016
  • Toxoplasma gondii strain FOU – Genbank version Aug 13, 2014
  • Toxoplasma gondii strain GAB2-2007-GAL-DOM2. – Genbank version Aug 13, 2014
  • Toxoplasma gondii strain MAS – Genbank version Aug 15, 2014
  • Toxoplasma gondii strain p89 – Genbank version Aug 13, 2014
  • Toxoplasma gondii strain RUB – Genbank version Aug 13, 2014
  • Toxoplasma gondii strain VAND – Genbank version Aug 15, 2014

Proteomics

  • Tachyzoite Ubiquitome – T. gondii ME49 tachyzoite ubiquitinated proteome. Silmon de Monerri et al.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary. Changes in annotation and new experimental data may slightly alter your search results by increasing or decreasing the number of hits.
