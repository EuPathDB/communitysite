We are pleased to announce the release of ToxoDB 35

New Data in Release 35

RNA Sequence

  • T gondii Transcriptome during infection in 4 mouse cell types — mRNA Illumina sequences from 4 different mouse cell types infected with T. gondii and aligned to the ME49 Genome as mixed parasite/host samples. Swierzy et al. This data set has a matching host data in HostDB — Transcriptomes of 4 M. musculus cell types during infection with T. gondii.

Proteomics

  • Extracellular vesicles (RH) — Toxoplasma gondii RH extracellular vesicle proteomics Wowk et al.

New features in this release

Dedicated search for Genes with protein products that have post-translational modifications

ToxoDB offers a new search, Identify Genes based on Post-Translational Modifications, which returns genes whose protein products were experimentally determined to be post-translationally modified. This search is categorized under Search for Genes, Proteomics, Post-Translational Modification.

GO Slim Implementation

GO Slim terms are now indicated in EuPathDB sites in the following ways:
  • The search, Identify Genes based on GO Terms, now includes an GO Subset option for limiting the search to only GO Slim generic terms.
  • GO Terms tables on gene pages now contain a 'GO Subset' column which identifies GO Slim generic terms.
  • The GO enrichment analysis has an option for limiting the analysis to GO Slim terms. This will limit both the background and the gene list of interest in the Fisher's exact test that determines enrichment.

Revigo and Word Cloud access from GO Enrichment results

ToxoDB offers a link between our GO Enrichment results and a Word Cloud visualization as well as Revigo, a tool that provides GO term summarization and visualization. The links to Revigo and the Word Cloud now appear as buttons that are right justified above the GO enrichment result table.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary.
