We are pleased to announce the release of ToxoDB 37

New Data in ToxoDB 37

Proteomics

  • Mitochondrial Matrix Quantitative Proteome — Toxoplasma mitochondrial matrix proteome determined using two independent, biotin-labelling approaches: ascorbate peroxidase (APEX) approach and the BirA*/BioID approach. This data was provided pre-publication by Giel van Dooren, Australian National University.
  • Mitochondrial Matrix Proteome — Toxoplasma mitochondrial matrix proteome determined using two independent, biotin-labelling approaches: ascorbate peroxidase (APEX) approach and the BirA*/BioID approach. Peptides are aligned to T gondii ME49. This data was provided pre-publication by Giel van Dooren, Australian National University.
  • Toxoplasma gondii RH N-terminal peptides from the Soldati-Favre lab, University of Zurich. Dogga et al. —
    Aligned to Toxoplasma gondii ME49
    Aligned to Toxoplasma gondii GT1

New features in this release

Beta EuPathDB Workspace

  • Announcing the Beta release of EuPathDB Workspace, a private workspace that can interface with EuPathDB Galaxy and allows for viewing and interrogating private data against data that is already integrated into EuPathDB (public data). The first EuPathDB workspace tool, 'Bigwig files to EuPathDB' exports bigwig files from EuPathDB Galaxy to EuPathDB. The tool creates a EuPathDB Data Set which is managed from the My Data Sets page along with your other private data sets. Each data set also receives a record page which lists all files that the data set contains, offers file management functions and ports files to the genome browser for viewing.

    Video tutorial: Export Bigwig files to EuPathDB

Pathway Improvements

  • Semantic zooming of Pathway drawings on Pathway record pages - At low zoom levels, pathway nodes appear as glyphs. At mid zoom levels, the main compound nodes appear as molecular formulas. At high zoom levels, enzymes and side nodes appear as formulas, terms or Enzyme commission numbers. Clicking on a node will retrieve details about the node, regardless of zoom level.
