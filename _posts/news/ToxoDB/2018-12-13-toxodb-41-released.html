We are pleased to announce the release of ToxoDB 41

New data in ToxoDB 41

Transcriptomics - Microarray

  • PMRT1 KO, WT and KOCM expression profiles — Expression profiles of arginine methyltransferase 1 (PMRT1) knockout, complement and wild type T. gondii RH-hxgprt parasites determined by Affymetrix microarray (Toxo Gene chip) El Bissati et al.

Whole genome sequencing of isolate for SNP analysis

  • Toxoplasma gondii strain CZ clone H3 aligned genome sequence — Genomic DNA sequencing from Toxoplasma gondii strain CZ clone H3. Data were used to call SNPs and determine CNV. Data were provided by the Heil Lab, University of Zurich.

New features in ToxoDB 41

Introducing Clustal Omega


EuPathDB sites are now using Clustal Omega for gene page protein alignments in the 'Orthologs and Paralogs within EuPathDB' table and the 'Retrieve multiple sequence alignment or multi-FASTA' tool in the 'Orthology and synteny' section of gene pages. The 'Isolate Alignments in this Gene Region' tool still uses CulstalW.

New Gene page tools

  • Interpro —
    The 'Protein features and properties' section now includes an option to submit the protein sequence for sequence analysis and classification via InterPro.
  • GPI anchor prediction —
    The 'Protein features and properties' section now includes an option to submit the protein sequence for GPI modification site prediction via the GPI Lipid Anchor Project.


Note: Our bimonthly database releases incorporate new data and correct errors in old data when necessary.
