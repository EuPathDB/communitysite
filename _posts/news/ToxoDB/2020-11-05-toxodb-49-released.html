New data in {{project}} 49

EBI Pipeline

{{project}} 49 represents the first release that employs the EBI pipeline for analyzing genomes and RNA-Seq data. ({{project}} Analysis Methods) All genomes were run through the EBI pipeline, updating all analyses such as InterproScan and repeat identification. While much of the EBI pipeline is the same as our previous in-house pipeline, these differences are noted:
  • Expression values calculated from RNA-Seq data are now presented as TPM (Transcripts Per Kilobase Million) instead of FPKM (Fragments Per Kilobase Million) or RPKM (Reads Per Kilobase Million). Learn more
  • Signal peptide predictions are now made with SignalP-5.0, updated from SignalP-3.0.

Annotated genome

  • Besnoitia besnoiti strain Bb-Ger1 – PacBio sequence assembled with HGAP v3. Schares et al. 2017.

Unannotated genome sequence

  • Toxoplasma gondii strain RH 2016 – Full genome sequence for Toxoplasma gondii strain RH 2016 - sequenced with Illumina methods and assembled with SOAPdenovo v. May-2013. Lau et al. 2016.

Transcriptomics

  • Tachyzoite and Cyst RNA-seq – RNA sequencing of Besnoitia besnoiti Lisbon14 tachyzoite and cyst transcriptomes. Adrian Hehl labs.

RFLP Analysis

  • A canine isolate of Toxoplasma gondii – RFLP genotype of an isolate of Toxoplasma gondii obtained from a dog with neurological disease in the city of Salvador, Brazil.

New features in {{project}} 49

  • When revisiting a search page, your previous parameter values are automatically loaded. This is especially useful for the BLAST search which now retains the input sequence.
  • The cytoscape drawing on metabolic pathway record pages now contains:
    • a new search function for finding nodes within the drawing. After running a search, the drawing centers on the nodes which match the search term and highlights them with a bold green border.
    • a "Reset Layout" button that will clear all node paintings as well as reset the node positions and zoom levels.
    • a checkbox for toggling your mouse function between zooming the drawing or scrolling the page.
  • Data set and Organism record pages now contain direct links to our Downloads section where the option is available to download genome-scale sequence and annotation files.
  • The Identify RFLP Genotype Isolates based on RFLP Genotype' search now employs our 'filter parameter' for configuring the RFLP genotype of isolates returned by the search. Running the search with default parameters returns all RFLP genotyped isolates. Use the filter parameter to define the profile of genetic markers for isolates returned by the search.
