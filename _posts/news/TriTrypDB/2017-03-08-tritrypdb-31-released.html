We are pleased to announce the release of TriTrypDB 31

New data in TriTrypDB 31

Annotated Genome

Blechomonas ayalai strain B08-376 — The genome sequence and annotation of trypanosomatid Blechomonas ayalai strain B08-376 that was isolated from a flea (Siphonaptera), appears for the first time in TriTrypDB. This data was provided by the Yurchenko lab.

Updated Functional Annotation

Functional annotations in TriTrypDB 31 have been updated to GeneDB's January 2017 version for the following genomes:
TriTrypDB Genome Version Download GeneDB GFF file Genes Updated
Trypanosoma brucei 927 Tbruceibrucei927.noseq.gff3.gz 930 genes


Summary of annotation updates which include products, GO terms, EC numbers, gene names, synonyms and PMIDs.

Updated Structural Annotations from GeneDB

Structural annotations (gene models) in TriTrypDB 31 have been updated to GeneDB's January 2017 version for the following genomes:
TriTrypDB Genome Version Download GeneDB GFF file Genes Updated
Trypanosoma brucei 927 Tbruceibrucei927.noseq.gff3.gz 87 genes


Summary of annotation updates which include added and deleted genes, changes in CDS coordinates and updates in gene IDs.

In addition, poly A sites for Trypanosoma brucei 927 have been remapped and 3' UTR predictions fixed in this release.
