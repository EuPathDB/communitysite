New features in TriTrypDB 39

New Phenotype Searches

  • The Identify Genes based on Phenotype Evidence page now contains two new searches: 1. The Curated Phenotype (CP) search returns genes based on phenotype characteristics. For data sets without Phenotype Qualifier data, the CP search incorporates a filter parameter for quickly choosing the desired phenotype characteristics. The new Phenotype Text (PT) search returns genes whose phenotypic descriptions contain your text term.

FPKM Floor parameter in RNA-Seq fold change searches

  • RNA Seq fold change searches now offer an FPKM Floor parameter for adjusting the FPKM value of low expressing genes. To avoid misleading fold change values generated by dividing with very small numbers, we set very low FPKM values to a default FPKM floor which represents 10 mapped reads. The new parameter offers a range of floor values for raising or lowering the expression threshold of the search depending on the user's objectives. For more information use the parameter help (blue question mark icon) on the search page or read the detailed RNA seq search help.
