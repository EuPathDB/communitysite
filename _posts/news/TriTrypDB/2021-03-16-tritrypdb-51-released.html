New data in {{project}} 51

Cellular Localization and Imaging

TrypTag beta release of complete dataset
  • Cellular Localization of tagged T. brucei proteins – TrypTag is a project aiming to determine where every trypanosome protein localises within the cell. TrypTag data on TriTrypDB 51 now reflects the first beta release of the complete dataset from the TrypTag project. TriTrypDB 51 includes additional images and localization information associated with both already existing and newly obtained GO terms, bringing the total number of genes with TrypTag images to 7,487. Dean et al. 2017 and Halliday et al. 2019

Summary of annotation updates which include products, GO terms, gene names, synonyms, publications, external database references and notes from curators. These functional annotations are current with GeneDB January, 2021 GFF versions.

New features in {{project}} 51

  • BLASTbeta tool – a new BLAST tool with multi-BLAST capabilities. Input a single sequence or several FASTA-formatted sequences at once. Our tool uses NCBI BLAST to find genes, genomic sequences, ESTs, or PopSet Sequences that have BLAST similarity to your input sequence. BLASTbeta video introduction (4 min)
  • Usage and performance metrics for VEuPathDB sites are now updated monthly.
