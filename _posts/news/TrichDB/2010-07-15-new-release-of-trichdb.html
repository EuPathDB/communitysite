--- layout: post title: New Release of TrichDB category: [EuPathDB,VEuPathDB,TrichDB] --- We are pleased to announce a new release of TrichDB. While this release does not include new data, there are some new features and improvements.

  • The Basket and Favorites functionality has been extended to include all record classes.
  • Queries have been optimized to increase site performance.
  • A number of bugs have been fixed.
