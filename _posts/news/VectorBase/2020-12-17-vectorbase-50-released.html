We are pleased to announce the release of {{project}} 50.

The URL https://{{project}}.org directs you to the newest version of our interface. The site contains previous VectorBase.org data plus some new tools and functions in an updated and streamlined interface that empasizes easy access to help information. Navigate to https://legacy.{{project}}.org for the previous version of the interface. The legacy interface will remain active for at least one more release.

Content migration from legacy to beta VectorBase

We continue to migrate all the content from the legacy site before it is shut down, next on our list are the proteome datasets and the Image Gallery. Please let us know which tool, resource, or dataset we might be missing or needs to be prioritized for migration.

Omics datasets

  • In VectorBase we have a total of 327 RNA-seq datasets. For the previous release, 96 RNAseq datasets were available in the search strategy system. For the current release we now have a total of 113 RNAseq datsets; more datasets will be included in following releases.

New features in {{project}} 50

  • OrthoMCL DB now shares web architecture with other VEuPathDB sites. The updated and streamlined interface emphasizes easy access to tools and help information.
    • OrthoMCL now includes orthology information for 564 species, including all of the VEuPathDB organisms that have predicted in silico proteomes.
    • The redesigned home page includes a new site-wide search for easily finding groups, proteins or documents. A categorized list of Searches appears on the left of the home page and the center section is devoted to tutorials for getting you started with OrthoMCL features.
    • Group and protein record pages are redesigned and include a contents section for navigating to the data you are interested in.
    • For more information see the OrthoMCL DB About page, the FAQ page or complete this tutorial.
  • New Community Related Sites: A few new links have been added to the Related Sites list. These include the DRSC/TRiP Functional Genomics Resources. All the resource available tools are listed here, including CRISPR GuideXpress.
