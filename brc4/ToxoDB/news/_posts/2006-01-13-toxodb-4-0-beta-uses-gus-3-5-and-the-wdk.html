--- layout: post title: ToxoDB 4.0 Beta uses GUS 3.5 and the WDK category: [ToxoDB] --- ToxoDB 4.0 Beta has been upgraded to use the latest Genomics Unified Schema (GUS) database system and the new Web Development Kit (WDK). GUS is an open source project (www.gusdb.org) providing a functional genomics database schema covering sequence annotation, expression data, regulatory interactions, and proteomics using controlled vocabularies and ontologies. GUS also provides software to load data (plugins) and set up a web site (WDK). GUS has been adopted by several groups covering organism ranging from bacteria, protozoa, plants, and mammals.
