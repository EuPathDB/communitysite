STRING <a href="http://www.ncbi.nlm.nih.gov/pubmed/25352553">(PMID:25352553)</a> protein-protein interaction database cross references have been added for <i>Ae. aegypti</i> (9333), <i>An. darlingi</i> (4555), <i>An. gambiae</i> (7872), <i>Cx. quinquefasciatus</i> (11109) and <i>Ixodes scapularis</i> (9349). STRING records can be viewed either via the transcript records for each gene in the genome browser  (e.g. <a href="/Anopheles_gambiae/Transcript/Similarity?g=AGAP004707;r=2L:2358158-2431617;t=AGAP004707-RA">AGAP004707-RA</a>), or retrieved in bulk via BioMart.
