How to run a structured search using web services

Overview

PlasmoDB provides programmatic access to its searches, via REST Web Services. The result of a web service request is a list of records (genes, compounds, etc.) in one of various formats (json, csv, etc.). REST services can be executed in a browser by typing a specific URL.

For example, this URL:
https://plasmodb.org/plasmo/service/record-types/transcript/searches/GenesByMolecularWeight/reports/standard?organism=Plasmodium falciparum3D7&min_molecular_weight=10000&max_molecular_weight=10500&reportConfig={"attributes":["gene_source_id","source_id","organism","gene_type"]}

Corresponds to this request:
Find all (Plasmodium falciparum,Plasmodium knowlesi) genes that have molecular weight between 10,000 and 50,000.
For each gene ID in the result, return its gene type and organism.
Provide the result in this format.

The Details

  1. First choose a search you want to run via web services. For example, you can go to the home page and select a search:
  2. Next fill in the parameters in the search page, to describe how you would like the web service URL parameters to be configured.
  3. When done filling in parameters, click the Build a web service URL from this search link. You will go to the Build a web services URL page:
  4. Choose a report, and then configure the report in the way you would like your web services URL to be configured.
  5. Finally, click the Build URL button. Your built up URL will display in the box below. It is ready for use.


Downloading DNA sequences in a text file in FASTA format

For specific genomic segments:
  • To download one sequence, please use one of the following URL formats:
    http://PlasmoDB.org/cgi-bin/contigSrt?project_id=PlasmoDB&ids=Pf3D7_04_v3&start=14&end=700
    http://PlasmoDB.org/cgi-bin/contigSrt?project_id=PlasmoDB&ids=Pf3D7_04_v3:14..700
  • For multiple sequences use the line feed character (%0A) as separator (comma or semicolon or carriage return do not work):
    http://PlasmoDB.org/cgi-bin/contigSrt?project_id=PlasmoDB&ids=Pf3D7_04_v3:14..700%0APf3D7_04_v3:800..900
For gene related regions:
  • To download one sequence, please use one of the following URL format:
    http://PlasmoDB.org/cgi-bin/geneSrt?project_id=PlasmoDB&ids=PF3D7_1133400&type=genomic&upstreamAnchor=Start&upstreamSign=minus&upstreamOffset=10&downstreamAnchor=End&downstreamSign=plus&downstreamOffset=2000
