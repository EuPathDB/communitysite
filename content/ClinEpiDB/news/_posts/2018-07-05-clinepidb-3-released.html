We are pleased to announce the release of ClinEpiDB 3

New Features in this Release

  • A new tutorial explains the three analysis apps available in ClinEpiDB after returning search results. These include distributions, contingency tables, and data summarization statistics.
  • For each study that will be represented in ClinEpiDB there will be a detailed study data set page. Here you will find information about the study that will be necessary to accurately interpret the data including key details like study design, data collection methodology and a description of the population of interest. For PRISM, in this release we have added data collection materials (Case Report Forms) and data dictionaries.

No new Data in this Release
